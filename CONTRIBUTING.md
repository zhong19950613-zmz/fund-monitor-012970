# Contributing to fund-monitor

Thank you for your interest in contributing to fund-monitor!

## How to Contribute

1. **Report bugs** — Use the Bug Report issue template
2. **Request features** — Use the Feature Request issue template
3. **Submit code** — Fork the repo, create a feature branch, and open a Pull Request

## Development Setup

```bash
git clone https://github.com/zhong19950613-zmz/fund-monitor-012970.git
cd fund-monitor-012970

# Run analysis locally
python scripts/analyze.py
```

## Code Style

- Keep scripts simple and well-commented
- Use meaningful variable names
- Update `data/holdings.yaml` structure carefully

## Commit Messages

Please use conventional commits:
- `feat:` new feature
- `fix:` bug fix
- `docs:` documentation
- `chore:` maintenance
